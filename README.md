# 最新英美剧集实时更新

一个专注于英美剧集信息的聚合展示平台，自动从 TMDB 收集最新美剧、英剧的上映信息，并整合豆瓣评分数据。项目每日自动更新，提供多维度筛选功能，帮助用户快速发现和追踪感兴趣的剧集。支持按地区、评分、流媒体平台、类型等多条件组合筛选，配备交互式年份时间线导航，让剧集浏览更加便捷高效。

在线体验：https://stevenspage.github.io/latest_tv

## 功能特性

### 🎬 多维度筛选
- **地区筛选**：支持美剧和英剧切换
- **评分筛选**：按豆瓣评分筛选（全部、>9分、>8分、>7分、近2年高分）
- **流媒体平台筛选**：支持 Netflix、Apple TV、Hulu、Prime Video、Disney、Paramount、HBO、ABC、FOX、CBS、NBC 等主流平台
- **类型筛选**：支持剧情、喜剧、悬疑、犯罪、科幻等多种类型筛选
- **多选筛选**：支持同时选择多个类型和平台进行组合筛选

### 📅 时间线导航
- **交互式年份时间线**：快速跳转到指定年份的剧集
- **按月份分组显示**：剧集按首播月份自动分组展示
- **即将上映专区**：单独展示即将上映的剧集，支持横向滚动浏览

### 🎨 界面特性
- **响应式设计**：适配各种屏幕尺寸
- **骨架屏加载**：优雅的加载动画，提升用户体验
- **粘性筛选栏**：类型筛选栏固定在顶部，方便随时切换
- **剧集卡片展示**：精美的卡片式布局，展示剧集海报、名称、评分等信息

### ⚡ 性能优化
- **快速初始加载**：优先加载最新数据，快速展示内容
- **后台数据加载**：完整数据在后台异步加载，不影响初始体验
- **数据缓存机制**：美剧和英剧数据分别缓存，切换更流畅
- **分页加载**：支持分页加载，提升大数据量下的性能

### 📊 数据展示
- **实时更新日期**：显示数据最后更新时间
- **豆瓣评分显示**：展示剧集的豆瓣评分和链接
- **详细信息**：显示剧集的首播日期、平台、类型等完整信息


## 数据来源
- 剧集信息：The Movie Database (TMDB)
- 评分信息：豆瓣
